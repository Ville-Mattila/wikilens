Wiki·Lens

/ˈwɪk.i lenz/ · noun, browser extension

  1. A Chromium extension that reveals the Wikipedia article behind any text you select — an image, the opening paragraph, and a link to read on, in a small popup right where you are.
  2. figurative. The shortest distance between curiosity and knowledge.

§ 2

Character­istics

fig. aExact by default

A preview appears only when your selection is precisely an article's title. Prefer looser matching? Flip one switch and redirects like “USA” resolve too.

fig. b23 languages

Point it at any Wikipedia edition — English, Suomi, Deutsch, 日本語 and nineteen more — from the settings page.

fig. cYours to shape

Three popup sizes, light and dark themes, and the full first paragraph in a scrollable card.

fig. dPrivate by design

Your selection travels to Wikipedia's API and nowhere else. Nothing is stored, logged, or sold. The code is open for inspection.

§ 3

Installation

  1. Download or clone the source from github.com/Ville-Mattila/wikilens.
  2. Open chrome://extensions and enable Developer mode.
  3. Click Load unpacked and choose the wikilens folder.
  4. Select some text anywhere. That's it.
